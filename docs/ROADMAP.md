# 🗺 Rocket-Leaf 路线图

本文档描述了 Rocket-Leaf 项目的功能规划和开发阶段。

---

## 开发阶段概览

| 阶段  | 名称       | 核心目标               |   状态   |
| :---: | ---------- | ---------------------- | :------: |
|   1   | 基础框架   | 搭建应用框架与基础 UI  | 🚧 进行中 |
|   2   | 连接管理   | 实现 RocketMQ 连接配置 | 📋 规划中 |
|   3   | Topic 管理 | Topic 的增删查改       | 📋 规划中 |
|   4   | 消费者组   | 消费者组管理与监控     | 📋 规划中 |
|   5   | 消息管理   | 消息查询与发送         | 📋 规划中 |
|   6   | 监控告警   | 集群监控与告警         | 📋 规划中 |

---

## Phase 1: 基础框架 🚧

**目标**：完成项目基础架构搭建，实现核心 UI 布局

### 功能清单

- [x] 项目初始化 (Wails3 + Vue3 + TypeScript)
- [ ] 集成 Naive UI 组件库
- [ ] 实现应用主体布局
  - [ ] 顶部标题栏（支持窗口拖拽、最小化/最大化/关闭）
  - [ ] 左侧导航栏（连接列表、功能菜单）
  - [ ] 主内容区域
- [ ] 主题系统
  - [ ] 明亮/暗黑主题切换
  - [ ] 跟随系统主题
- [ ] 国际化基础设施（中/英文支持）

### 技术要点

- 使用 Naive UI 的 `n-layout` 实现响应式布局
- 使用 CSS 变量实现主题切换
- 使用 Vue I18n 实现国际化

---

## Phase 2: 连接管理 📋

**目标**：实现 RocketMQ 连接的配置与管理

### 功能清单

- [ ] 连接配置表单
  - [ ] 连接名称
  - [ ] NameServer 地址（支持多个）
  - [ ] AccessKey / SecretKey（可选）
  - [ ] 连接超时设置
- [ ] 连接列表管理
  - [ ] 添加新连接
  - [ ] 编辑已有连接
  - [ ] 删除连接
  - [ ] 快速切换连接
- [ ] 连接状态
  - [ ] 连接测试
  - [ ] 实时状态检测
  - [ ] 断线自动重连
- [ ] 数据持久化
  - [ ] 本地存储连接配置
  - [ ] 敏感信息加密存储

### 技术要点

- Go 端使用 `rocketmq-admin-go` 库与 RocketMQ 交互
- 使用 SQLite 或 JSON 文件存储连接配置
- 敏感信息使用 AES 加密
